# Asteroids

A simple yet fun Asteroids game built with Python and Pygame. Navigate your ship through space, dodge asteroids, and blast them to pieces!

## Features

- Classic arcade-style gameplay
- Player-controlled spaceship
- Randomly spawning asteroids
- Projectile shooting mechanics

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/edwinboon/asteroids.git
   ```
2. Navigate into the project directory:
   ```sh
   cd asteroids
   ```
3. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
4. Run the game:
   ```sh
   python main.py
   ```

## Controls

- **Arrow Keys**: Move the spaceship
- **Spacebar**: Shoot
- **Esc**: Quit the game
